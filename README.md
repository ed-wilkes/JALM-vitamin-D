# JALM-vitamin-D
Code and data for Bayesian modelling of vitamin D stability data.

`Wilkes_JALM_stability_data.csv` represents the freely-available data from the below publication:

```
@article{sempos2022,
    title={Assessment of serum total 25-hydroxyvitamin D assays for Vitamin D External Quality Assessment Scheme (DEQAS) materials distributed at ambient and frozen conditions},
    author={Sempos CT et al.},
    journal={Anal Bioanal Chem},
    year={2022}
}
```
